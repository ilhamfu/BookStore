# Bookstore

Tugas kecil praktikum pemrograman web tahun 2018

 
# Fitur:
    -Login dan Daftar
    -Admin diberi hak untuk
        -menambah buku
        -menghapus buku
        -menghapus user
        -memberi hak akses kepada user
        -menyatakan suatu buku telah di kembalikan
    -User dapat
        -meminjam buku
        -mengecek buku yang di pinjam
